# Kanban Board

Простая и функциональная канбан-доска для управления задачами, созданная на чистом HTML, CSS и JavaScript.

## Описание

Это веб-приложение представляет собой классическую канбан-доску с тремя колонками для отслеживания прогресса задач. Приложение работает полностью в браузере без необходимости установки дополнительного ПО или сервера.

## Возможности

- **Три колонки**: To Do, In Progress, Done
- **Создание задач**: Добавление новых карточек в любую колонку
- **Редактирование**: Изменение названия и описания существующих задач
- **Удаление**: Удаление ненужных задач с подтверждением
- **Перемещение**: Перемещение карточек между колонками с помощью кнопок
- **Автосохранение**: Все данные сохраняются в localStorage браузера
- **Адаптивный дизайн**: Работает на desktop, планшетах и мобильных устройствах
- **Счетчики**: Отображение количества задач в каждой колонке

## Установка и запуск

1. Скачайте или клонируйте файлы проекта:
   ```bash
   git clone <repository-url>
   cd kanban
   ```

2. Откройте файл `index.html` в браузере:
   - Дважды кликните на `index.html`, или
   - Перетащите файл в окно браузера, или
   - Используйте команду:
     ```bash
     open index.html  # macOS
     start index.html # Windows
     xdg-open index.html # Linux
     ```

3. Готово! Приложение работает.

## Использование

### Создание новой задачи

1. Нажмите кнопку **"+ Добавить карточку"** в нужной колонке
2. Введите название задачи (обязательно)
3. Добавьте описание (опционально)
4. Нажмите **"Сохранить"**

### Редактирование задачи

1. Нажмите кнопку **"Редактировать"** на карточке
2. Измените название или описание
3. Нажмите **"Сохранить"**

### Перемещение задачи

- Используйте кнопки **"← Назад"** или **"Далее →"** на карточке
- Кнопки автоматически отключаются на крайних позициях

### Удаление задачи

1. Нажмите кнопку **"Удалить"** на карточке
2. Подтвердите удаление в диалоговом окне

### Горячие клавиши

- **Escape** - закрыть модальное окно

## Технические детали

### Технологии

- HTML5 (семантическая разметка)
- CSS3 (Flexbox, transitions, responsive design)
- Vanilla JavaScript (ES6+)

### Браузеры

Приложение работает во всех современных браузерах:
- Chrome/Edge (последние версии)
- Firefox (последние версии)
- Safari (последние версии)
- Opera (последние версии)

### Хранение данных

Все задачи сохраняются в **localStorage** браузера:
- Данные сохраняются автоматически после каждого изменения
- Данные загружаются при открытии приложения
- Данные привязаны к домену и браузеру
- Очистка данных браузера удалит все задачи
